No More Special Days

那些人，那些事，那些过往
那些喧嚣，那些悸动，那些怯懦
那些幸福，那些感动，那些怨恨
那些付出，那些索取，那些麻木

no more special days,
every day is an ordinary day,
like today, like yesterday, like these days to come,
like these days will never come
